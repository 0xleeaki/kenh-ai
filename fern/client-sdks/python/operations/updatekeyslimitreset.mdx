New limit reset type for the API key (daily, weekly, monthly, or null for no reset). Resets happen automatically at midnight UTC, and weeks are Monday through Sunday.

## Values

| Name      | Value     |
| --------- | --------- |
| `DAILY`   | daily     |
| `WEEKLY`  | weekly    |
| `MONTHLY` | monthly   |
