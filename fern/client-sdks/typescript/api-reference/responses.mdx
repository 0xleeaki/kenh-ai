## Overview

beta.responses endpoints

### Available Operations

* [send](#send) - Create a response

## send

Creates a streaming or non-streaming response using OpenResponses API format

### Example Usage: guardrail-blocked

{/* UsageSnippet language="typescript" operationID="createResponses" method="post" path="/responses" example="guardrail-blocked" */}
```typescript lines
import { KenhAI } from "@kenh-ai/sdk";

const kenhAi = new KenhAI({
  httpReferer: "<value>",
  appTitle: "<value>",
  appCategories: "<value>",
  apiKey: process.env["KENH_AI_API_KEY"] ?? "",
});

async function run() {
  const result = await kenhAi.beta.responses.send({
    responsesRequest: {},
  });

  console.log(result);
}

run();
```
