## Overview

Credit management endpoints

### Available Operations

* [getCredits](#getcredits) - Get remaining credits

## getCredits

Get total credits purchased and used for the authenticated user. [Management key](/guides/overview/auth/management-api-keys) required.

### Example Usage

{/* UsageSnippet language="typescript" operationID="getCredits" method="get" path="/credits" */}
```typescript lines
import { KenhAI } from "@kenh-ai/sdk";

const kenhAi = new KenhAI({
  httpReferer: "<value>",
  appTitle: "<value>",
  appCategories: "<value>",
  apiKey: process.env["KENH_AI_API_KEY"] ?? "",
});

async function run() {
  const result = await kenhAi.credits.getCredits();

  console.log(result);
}

run();
```

  appCategories: "<value>",

run();
