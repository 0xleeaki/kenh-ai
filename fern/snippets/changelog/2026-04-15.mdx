### Product changes

- **Regex prompt injection detection** -- Guardrails pipeline now includes regex-based prompt injection detection for an additional layer of security. See [Guardrails docs](/guides/features/guardrails).
- **Activity explorer top-N control** -- The activity explorer now has an adjustable top-N control with a stepper UI and "Show Other" toggle for more flexible data visualization.
- **Auto Router pattern warnings** -- The auto-router UI now shows warnings for individual model patterns that don't match any supported models.
- **Clickable job rows** -- Rows in the [Jobs](https://kenh-ai.com/logs?tab=jobs) table are now clickable to open the job detail view directly.
