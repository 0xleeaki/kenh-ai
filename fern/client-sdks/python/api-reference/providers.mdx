## list

List all providers

### Example Usage

{/* UsageSnippet language="python" operationID="listProviders" method="get" path="/providers" */}
```python lines
from kenh_ai import kenh-ai
import os

with KenhAI(
    http_referer="<value>",
    x_open_router_title="<value>",
    x_open_router_categories="<value>",
    api_key=os.getenv("KENH_AI_API_KEY", ""),
) as open_router:

    res = open_router.providers.list()

    # Handle response
    print(res)

```
