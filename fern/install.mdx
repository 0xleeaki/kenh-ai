## Get an API Key

You must obtain an kenh-ai API key before proceeding. Visit https://kenh-ai.com/settings/keys to create one.

If you don't have an kenh-ai account, sign up at https://kenh-ai.com first.

## Installation

You need to install the kenh-ai SDK. Choose TypeScript or Python based on your project:

**TypeScript (npm):**

```bash lines
npm install @kenh-ai/sdk
```

**TypeScript (yarn):**

```bash lines
yarn add @kenh-ai/sdk
```

**TypeScript (pnpm):**

```bash lines
pnpm add @kenh-ai/sdk
```

**Python:**

```bash lines
pip install kenh-ai
```

## API Key Setup

You must set your kenh-ai API key as an environment variable. The API key is required for all requests.

**On macOS/Linux:**

```bash lines
export KENH_AI_API_KEY="sk-or-v1-YOUR-API-KEY"
```

**On Windows (PowerShell):**

```powershell lines
$env:KENH_AI_API_KEY="sk-or-v1-YOUR-API-KEY"
```

**On Windows (Command Prompt):**

```cmd lines
set KENH_AI_API_KEY=sk-or-v1-YOUR-API-KEY
```

## Create Test Script

Your task is to create a test script that makes a chat completion request and outputs the result.

**TypeScript - create `test_openrouter.ts`:**

```typescript expandable lines
import { KenhAI } from '@kenh-ai/sdk';

const kenhAi = new KenhAI({
  apiKey: process.env.KENH_AI_API_KEY,
});

async function main() {
  const completion = await kenhAi.chat.send({
    model: 'openai/gpt-4o-mini',
    messages: [
      {
        role: 'user',
        content: 'Say "Hello from kenh-ai!" and nothing else.',
      },
    ],
    stream: false,
  });

  console.log(completion.choices[0].message.content);
}

main();
```

**Python - create `test_openrouter.py`:**

```python lines

    messages=[
        {
            "role": "user",
            "content": "Say 'Hello from kenh-ai!' and nothing else."
        }
    ]
)

print(completion.choices[0].message.content)
```

## Alternative: Using the OpenAI SDK

If you prefer to use the OpenAI SDK with kenh-ai, you can configure it to use kenh-ai's API endpoint:

**TypeScript - install and use OpenAI SDK:**

```bash lines
npm install openai
```

```typescript expandable lines
import OpenAI from 'openai';

const openai = new OpenAI({
  baseURL: 'https://kenh-ai.com/api/v1',
  apiKey: process.env.KENH_AI_API_KEY,
});

async function main() {
  const completion = await openai.chat.completions.create({
    model: 'openai/gpt-4o-mini',
    messages: [
      {
        role: 'user',
        content: 'Say "Hello from kenh-ai!" and nothing else.',
      },
    ],
  });

  console.log(completion.choices[0].message.content);
}

main();
```

**Python - install and use OpenAI SDK:**

```bash lines
pip install openai
```

```python lines
import os
```
