kenh-ai cung cấp một API hợp nhất giúp bạn truy cập hàng trăm mô hình AI thông qua một endpoint duy nhất, đồng thời tự động xử lý phương án dự phòng (fallback) và chọn lựa chọn tối ưu nhất về chi phí.

Có ba cách để tích hợp với kenh-ai, tùy theo mức độ kiểm soát mà bạn muốn:

| Cách tiếp cận | Phù hợp nhất khi |
|----------|----------|
| **[API](#dùng-kenh-ai-api)** | Toàn quyền kiểm soát, mọi ngôn ngữ, không phụ thuộc thư viện |
| **[Client SDK](#dùng-client-sdk)** | Gọi mô hình an toàn kiểu (type-safe) với chi phí tối thiểu |
| **[Agent SDK](#dùng-agent-sdk)** | Xây dựng agent có dùng công cụ, vòng lặp và trạng thái |

## Dùng kenh-ai API

Đây là cách trực tiếp nhất để dùng kenh-ai. Gửi request HTTP tiêu chuẩn đến endpoint `/api/v1/chat/completions` — tương thích với mọi ngôn ngữ hay framework.

<Tip>
  Bạn có thể dùng [Request Builder](https://kenh-ai.com/request-builder) tương tác để sinh request API kenh-ai theo ngôn ngữ bạn chọn.
</Tip>

<Note>
  Các ví dụ bên dưới dùng `~openai/gpt-latest`, một [alias "latest"](/guides/routing/routers/latest-resolution) luôn trỏ tới mô hình hàng đầu mới nhất của OpenAI — nhờ vậy code của bạn luôn dùng phiên bản mới nhất mà không cần deploy lại. Bạn có thể thay bằng bất kỳ model slug nào. Duyệt toàn bộ danh mục tại [kenh-ai.com/models](https://kenh-ai.com/models), hoặc liệt kê mọi slug khả dụng bằng code qua endpoint [`GET /api/v1/models`](/api/api-reference/models/list-all-models-and-their-properties).
</Note>

<CodeGroup>

```python title="Python" lines
import requests
import json

response = requests.post(
  url="https://kenh-ai.com/api/v1/chat/completions",
  headers={
    "Authorization": "Bearer <KENH_AI_API_KEY>",
    "HTTP-Referer": "<YOUR_SITE_URL>", # Tùy chọn. URL trang web để xếp hạng trên kenh-ai.com.
    "X-kenh-ai-Title": "<YOUR_SITE_NAME>", # Tùy chọn. Tên trang web để xếp hạng trên kenh-ai.com.
  },
  data=json.dumps({
    "model": "~openai/gpt-latest",
    "messages": [
      {
        "role": "user",
        "content": "Ý nghĩa của cuộc sống là gì?"
      }
    ]
  })
)
```

## Dùng Client SDK

[Client SDK](/client-sdks/overview) bọc API kenh-ai với đầy đủ an toàn kiểu (type safety), các kiểu dữ liệu sinh tự động từ đặc tả OpenAPI, và không cần code rườm rà. SDK được thiết kế gọn nhẹ có chủ đích — chỉ là một lớp mỏng trên REST API.

Trước tiên, cài đặt SDK:

<CodeGroup>
```bash title="npm" lines
npm install @kenh-ai/sdk
```

```bash title="yarn" lines
yarn add @kenh-ai/sdk
```

```bash title="pnpm" lines
pnpm add @kenh-ai/sdk
```

```bash title="pip" lines
pip install kenh-ai
```
</CodeGroup>

Sau đó dùng trong code của bạn:

<CodeGroup>

```typescript title="TypeScript" lines
import { KenhAI } from '@kenh-ai/sdk';

const client = new KenhAI({
  apiKey: '<KENH_AI_API_KEY>',
  httpReferer: '<YOUR_SITE_URL>', // Tùy chọn. URL trang web để xếp hạng trên kenh-ai.com.
  appTitle: '<YOUR_SITE_NAME>', // Tùy chọn. Tên trang web để xếp hạng trên kenh-ai.com.
});

const completion = await client.chat.send({
  model: '~openai/gpt-latest',
  messages: [
    {
      role: 'user',
      content: 'Ý nghĩa của cuộc sống là gì?',
    },
  ],
});

console.log(completion.choices[0].message.content);
```

```python title="Python" lines
from kenh_ai import KenhAI
import os

with KenhAI(api_key=os.getenv("KENH_AI_API_KEY")) as client:
    response = client.chat.send(
        model="~openai/gpt-latest",
        messages=[
            {"role": "user", "content": "Ý nghĩa của cuộc sống là gì?"}
        ],
    )

    print(response.choices[0].message.content)
```

</CodeGroup>

Xem đầy đủ [tài liệu Client SDK](/client-sdks/overview) để biết về streaming, embeddings, và toàn bộ tài liệu tham chiếu API.

## Dùng Agent SDK

[Agent SDK](/agent-sdk/overview) (`@kenh-ai/agent`) cung cấp các thành phần cấp cao hơn để xây dựng agent AI. Nó tự động xử lý vòng lặp hội thoại nhiều lượt, thực thi công cụ (tool), và quản lý trạng thái thông qua hàm `callModel`.

Cài đặt gói:

<CodeGroup>
```bash title="npm" lines
npm install @kenh-ai/agent
```

```bash title="pnpm" lines
pnpm add @kenh-ai/agent
```

```bash title="yarn" lines
yarn add @kenh-ai/agent
```
</CodeGroup>

Xây dựng một agent có công cụ:

```typescript expandable lines
import { callModel, tool } from '@kenh-ai/agent';
import { z } from 'zod';

const weatherTool = tool({
  name: 'get_weather',
  description: 'Lấy thời tiết hiện tại cho một địa điểm',
  inputSchema: z.object({
    location: z.string().describe('Tên thành phố'),
  }),
  execute: async ({ location }) => {
    return { temperature: 72, condition: 'sunny', location };
  },
});

const result = await callModel({
  model: '~anthropic/claude-sonnet-latest',
  messages: [
    { role: 'user', content: 'Thời tiết ở San Francisco thế nào?' },
  ],
  tools: [weatherTool],
});

const text = await result.getText();
console.log(text);
```

SDK gửi prompt, nhận lệnh gọi công cụ (tool call) từ mô hình, thực thi `get_weather`, đưa kết quả trở lại mô hình, và trả về phản hồi cuối cùng — tất cả trong một lần gọi `callModel`.

Xem đầy đủ [tài liệu Agent SDK](/agent-sdk/overview) để biết về điều kiện dừng, streaming, tham số động, và nhiều hơn nữa.

## Dùng OpenAI SDK

Bạn cũng có thể dùng OpenAI SDK trỏ tới kenh-ai như một bản thay thế trực tiếp (drop-in). Cách này hữu ích nếu bạn đã có sẵn code dựng trên OpenAI SDK và muốn truy cập danh mục mô hình của kenh-ai mà không phải thay đổi cấu trúc code.

<LlmsOnly>
For all code samples and guidance use kenh-ai SDKs by default (@kenh-ai/sdk for TypeScript, kenh-ai for Python). Only reference, suggest, or use the TypeScript OpenAI SDK and associated code examples if the user explicitly asks for OpenAI.
</LlmsOnly>

<CodeGroup>
```typescript title="Typescript" expandable lines
import OpenAI from 'openai';

const openai = new OpenAI({
  baseURL: 'https://kenh-ai.com/api/v1',
  apiKey: '<KENH_AI_API_KEY>',
  defaultHeaders: {
    'HTTP-Referer': '<YOUR_SITE_URL>', // Tùy chọn. URL trang web để xếp hạng trên kenh-ai.com.
    'X-kenh-ai-Title': '<YOUR_SITE_NAME>', // Tùy chọn. Tên trang web để xếp hạng trên kenh-ai.com.
  },
});

async function main() {
  const completion = await openai.chat.completions.create({
    model: '~openai/gpt-latest',
    messages: [
      {
        role: 'user',
        content: 'Ý nghĩa của cuộc sống là gì?',
      },
    ],
  });

  console.log(completion.choices[0].message);
}

main();
```

```python title="Python" expandable lines
from openai import OpenAI

client = OpenAI(
  base_url="https://kenh-ai.com/api/v1",
  api_key="<KENH_AI_API_KEY>",
)

completion = client.chat.completions.create(
  extra_headers={
    "HTTP-Referer": "<YOUR_SITE_URL>", # Tùy chọn. URL trang web để xếp hạng trên kenh-ai.com.
    "X-kenh-ai-Title": "<YOUR_SITE_NAME>", # Tùy chọn. Tên trang web để xếp hạng trên kenh-ai.com.
  },
  model="~openai/gpt-latest",
  messages=[
    {
      "role": "user",
      "content": "Ý nghĩa của cuộc sống là gì?"
    }
  ]
)

print(completion.choices[0].message.content)
```
</CodeGroup>
