### Product changes

- **Model compare benchmarks** -- The [model compare](https://kenh-ai.com/compare) page now includes Artificial Analysis and Design Arena benchmark scores for side-by-side evaluation.
- **Request caching for Messages API** -- The `/messages` endpoint now supports request caching. See [Response Caching docs](/guides/features/response-caching).
- **Sticky compare selectors** -- Model selectors on the compare page are now sticky with removal buttons for a smoother multi-model comparison experience.
