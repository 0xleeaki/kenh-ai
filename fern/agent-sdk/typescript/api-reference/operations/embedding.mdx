Embedding vector as an array of floats or a base64 string

## Supported Types

### `number[]`

```typescript lines
const value: number[] = [
  0.0023064255,
  -0.009327292,
  0.015797347,
];
```

### `string`

```typescript lines
const value: string = "<value>";
```
