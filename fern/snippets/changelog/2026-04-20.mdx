### Product changes

- **API key detail page** -- Clicking an API key now opens a dedicated detail page with usage stats, metadata, and a shareable URL.
- **Model picker hover panel** -- The model picker now shows a rich hover panel with title, icon, stats, and an uptime bar graph.
- **Text-to-speech audio download** -- Added a download button for text-to-speech audio output in the [Playground](https://kenh-ai.com/playground).
- **API key bulk actions toolbar** -- Replaced the bulk actions dropdown with a floating toolbar for faster key management.
- **Logs filter sorting** -- Your own email and API keys are now sorted to the top of filter lists on the [Activity](https://kenh-ai.com/activity) page.
- **I/O Logging API** -- Prompts and completions can now be retrieved via the API.
- **Web search pricing for Gemini** -- Default web search pricing is now shown for Gemini models with grounding support.
